## Installation

1. Go to the [Latest Release](https://github.com/TheNorthEestern/lagoon-releases/releases/latest) and download the DMG.
2. Open the DMG and drag **Lagoon Studio** to your `/Applications` folder.
3. Launch Lagoon Studio!

## Updates

Lagoon Studio includes automatic update support via [Sparkle](https://sparkle-project.org/). You'll be notified when new versions are available, or you can check manually via **Lagoon Studio > Check for Updates**.
